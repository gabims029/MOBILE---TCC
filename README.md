# Projeto Base - Mobile React Native

Este repositório fornece a estrutura inicial para o aplicativo mobile de gerenciamento de reservas de salas de aula. Ele se conecta à API desenvolvida em Node.js e permite que os usuários visualizem salas, agendem horários e gerenciem seus perfis.

## Objetivo da Sprint

- Implementar a interface mobile para interação com a API de reserva de salas de aula.
- Refatorar o código para separar lógicas e criar componentes reutilizáveis.
- Criar telas para login, cadastro, visualização de usuários, salas, agendamentos e períodos.
- Garantir autenticação e fluxo seguro de dados entre o app e a API.

## Pré-requisitos

- Node.js
- Dispositivo físico ou emulador Android/iOS
- API backend rodando (conforme o README da API)

## Instalação do Projeto Mobile

1. Clone o repositório:
   git clone https://github.com/gabims029/MOBILE---TCC.git
2. instalar npm i 
3. npx expo start 

### 1. Emulador Android/iOS
- Abra o Android Studio.
- Inicie um emulador.
- No Expo, clique em **Run on Android/iOS simulator** para abrir o app no emulador.

### 2. Dispositivo físico
- Instale o aplicativo **Expo Go** na Play Store (Android) ou App Store (iOS).
- No terminal, ao rodar `npx expo start`, aparecerá um QR code.
- Escaneie o QR code com o app Expo Go para abrir o projeto diretamente no celular.
